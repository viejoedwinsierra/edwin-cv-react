# CV Web de Edwin Gustavo Sierra Poveda
 
Proyecto React + Vite creado a partir de la información del CV/LinkedIn y con una estética inspirada en el formato 273 (dos columnas, panel lateral azul y contenido principal limpio).

## Cómo usar

```bash
npm install
npm run dev
```

## Compilar

```bash
npm run build
npm run preview
```

## Personalizaciones rápidas

- Edita `src/data.js` para cambiar textos, experiencia, skills o enlaces.
- Edita `src/styles.css` para colores, espaciado y estilo visual.
- El botón **Descargar / Imprimir CV** usa `window.print()` para exportar a PDF desde el navegador.

## Estructura

- `src/App.jsx`: layout principal
- `src/data.js`: contenido del CV
- `src/styles.css`: estilos
